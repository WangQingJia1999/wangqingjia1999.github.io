placement new

在预先申请好的一片内存上构造对象。

比如 vector 的 reserve()，先预先申请足够的内存，后续用 placement new按需构造对象。

在内存池和垃圾回收器中经常用到。

那有 placement delete 吗？

stl为什么用allocator而不是直接new？

因为stl需要将对象构造拆分为两步，1. 申请内存; 2. 构造对象。同样析构也拆分为，1. 析构对象; 2. 释放内存；如果没有这个拆分的需求，new足矣。
