内部链接与外部链接

内部链接（Internal Linkage）:

  1. 作用域：内部链接标识的符号仅在所定义的翻译单元可访问，其他翻译单元不可访问。
  2. 声明：
    1. 使用 static 关键字
    2. 非 extern 的 const
    3. 匿名命名空间内部所定义的符号为内部链接。
  3. 用途：
    • 限制作用域，避免命名冲突
    • 封装实现细节

外部链接（External Linkage）:

  1. 作用域：外部链接符号在整个程序的所有翻译单元内均可访问。
  2. 声明：
    1. 全局符号默认为外部链接链接。
    2. 使用 extern 关键字显式指定。符号定义到当前编译单元或其他链接的编译单元均可。
  3. 用途：
    • 多个翻译单元共享符号。
