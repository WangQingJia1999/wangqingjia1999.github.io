zset

Redis根据zset中元素的大小和数量，采用不同的存储方式，有如下两种：

  • listpack - 适用于小数据量情况，更省内存，增删改查 O(n)，以时间换空间
  • skiplist + dict - 适用于zset中某个元素较大或元素较多的情况，同时采用跳表和哈希表，充分利用了跳表 O(log) 范围查询(如ZRANGEBYSCORE操作)，和哈希表 O(1) 单点查询(如ZSCORE操作)的特征。
